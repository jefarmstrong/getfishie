Measure Help

Step: 0

Catch a fish!
Take a picture with the fish horizontal on the ground with some known object in the photo. In this case I'm using my Ugly Stik® GX2™ spinning rod.
To Measure Later - Tap the 'fish measure' button from the catch details.

Step: 1

I've already added the name and width of a portion of my Ugly Stik® (I'm using the distance between the two red lines which is 2.75 inches).

To add a new item, just tap the 'New Item Name' field and type in the name, then add the width of that item in inches in the next field.

NOTE: This item will be saved and next time will show up in the picker list.

Tap 'Set' when you're ready.

Step: 2

Now drag the two handles so that the vertical lines mark the left and right edges of the item. In this case I'm using the two red lines on the rod.

HINT: You can zoom into the photo to get a closer look using a pinch-out gesture on the photo.

Tap 'Set' when you're ready.

Step: 3

Now drag the two handles so that the vertical lines mark the left and right edges of the fish and you'll see the length and weight estimate change as you drag.

HINT: If the fish was moving it's tail or something, you can always add a little on one side to compensate.

Tap 'Set' when you're happy with everything to save these settings.

NOTE: This fish was actually weighed with a scale at 2.8 pounds - so Fishie Ruler can be quite accurate!
